# Project 3: Text Analyzer

> **Build text classification and generation systems using RNNs**

**Difficulty**: ⭐⭐ Intermediate  
**Time**: 3-4 hours  
**Prerequisites**: Steps 0-7 (Including RNN concepts)

---

## 🎯 Project Goals

Build text analysis systems:
1. **Text Classifier** - Classify text into categories
2. **Text Generator** - Generate new text character by character

---

## 📋 Problem 1: Text Classification

### Task
Classify text documents into categories (e.g., news categories, sentiment).

### Dataset
Create synthetic text data with:
- Document length
- Word frequencies
- Character patterns
- Category labels

### Requirements
- [ ] Preprocess text data
- [ ] Extract features or use embeddings
- [ ] Build RNN or feedforward network
- [ ] Train classifier
- [ ] Evaluate performance
- [ ] Classify new text

---

## 📋 Problem 2: Text Generation

### Task
Generate new text character by character using an RNN.

### Dataset
Use a simple text corpus (poems, quotes, etc.).

### Requirements
- [ ] Preprocess text (character-level)
- [ ] Build character-level RNN
- [ ] Train on text sequences
- [ ] Generate new text
- [ ] Experiment with temperature

---

## 🚀 Getting Started

See `text_classifier.py` and `text_generator.py` for implementations.
