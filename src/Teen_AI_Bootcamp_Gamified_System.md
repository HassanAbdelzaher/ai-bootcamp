# 🎮 Teen AI Bootcamp — Gamified Mission System
> Turn learning AI into a **game** with XP, levels, badges, and boss fights.

---

## 🧠 Core Philosophy
- Learning = Playing
- Progress = Visible
- Failure = Retry (no punishment)
- Mastery > Memorization

Students don’t *finish lessons* — they **level up**.

---

## ⭐ XP System

### Base XP Rules
| Action | XP |
|-----|----|
| Complete a mission | +100 XP |
| Bonus challenge | +30 XP |
| Explain concept clearly | +20 XP |
| Help another student | +10 XP |
| Clean & readable code | +20 XP |
| Visualization added | +20 XP |

⚠️ No negative XP. Mistakes are part of learning.
